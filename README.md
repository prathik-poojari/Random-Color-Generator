# JavaScript Practice Project

This repository contains a simple JavaScript project where I practiced generating random colors dynamically using JavaScript. It's a basic exercise aimed at improving my understanding of DOM manipulation and JavaScript functionality.

## Features

- Generates random RGB colors on button click.
- Utilizes JavaScript to dynamically change the background color of a div element.
- Responsive design with basic CSS styling.

## Demo

You can see a live demo of this project [here](https://prathik-poojari.github.io/Random-Color-Generator/).

## Getting Started

To get a copy of this project up and running on your local machine, follow these steps:

1. Clone the repository to your local machine using the following command:

    ```
    git clone https://github.com/your-username/javascript-practice.git
    ```

2. Open the project directory in your code editor of choice.

3. Explore the `index.html`, `style.css`, and `app.js` files to understand the structure and functionality of the project.

4. Open the `index.html` file in your web browser to see the project in action.

## Usage

To use the color generator:

1. Click the "Generate a Color" button.
2. The background color of the displayed div will change to a randomly generated color.
3. Each click generates a new random color.
